Lock Component

This is a simple control to set and show lock.

The control takes instance and index and displays the current lock state (using the stylesheet entries below). On pressing the button on this control the lock state is toggled..

ENHANCEMENT:

You can now use a comma deliminated list of indecies to perform a "Lock All" feature where one lock button locks and unlocks several destinations

The control will aggregate the lock status and only show locked or unlocked if all destinations are the same state

The text in the button will change from "LOCK" to "LOCK ALL" in this mode

Commands

Name Use
instance Normal instance targeting
index The index of on the lock device

Notifications

None

Stylesheets

Name Use
enum_unknown When a control is targeted but the value has not been returned from the device
dest_locked When the destination is locked (1)
dest_unlocked When the destination is not locked (0)
dest_mixlocked
 When  multiple destinations are grouped and they are in different lock states
