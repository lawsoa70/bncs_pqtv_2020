source_grid

Description

The Source Gird component gives and easy and repeatable way to give access to router mapping.

Commands

Name Type Use
instance designtime/runtime The router instance to use
layout designtime/runtime The layout to use
mapping designtime/runtime Allows mapping and name offsets to be set at design and runtime, comma delimited list of name=instance
page_mapoffset=,page_nameoffset=,grid_mapoffset=,grid_namesdb=
select_group designtime/runtime Select group, button id
select_page designtime/runtime Select page, button id
clear_selection designtime/runtime Clears the selected index

Notifications

Events

Name Use
index=<selected index> Notifies when the selected index changes
group_index=<selected group> Notifies when the group index changes
page_index=<selected page> Notifies when the page index changes
clear_selection Clears the index selection
