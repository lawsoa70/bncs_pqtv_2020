Detailed Description

This is a class to split up strings in <parameter>=<value> format.

ininput string

The maximum number of elements is SUBS, which is currently

Example:
decodeCaptionString str( "param.1.2.3=abc");
int count = str.subs; // count contains 3
printf( "command: %s", str.command ); // output "param"
printf( "first: %s", str.sub[ 0 ] ); // output "1"
printf( "second: %s", str.sub[ 1 ] ); // output "2"
printf( "third: %s", str.sub[ 2 ] ); // output "3"
printf( "value: %s", str.value ); // output "abc"
