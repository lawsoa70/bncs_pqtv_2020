Detailed Description

This is a class to split up strings in BNCS format. e.g. IR 123 001 001 'hello mum'.

ininput string

The maximum number of elements is BNCS_STRING_SUBS, which is currently 32

Example:
decodeBNCSString str( "IR 123 001 001 'hello mum'");
int count = str.subs; // count contains 5
printf( "first: %s", str.sub[ 0 ] ); // output "IR"
printf( "second: %s", str.sub[ 1 ] ); // output "123"
printf( "third: %s", str.sub[ 2 ] ); // output "001"
printf( "forth: %s", str.sub[ 3 ] ); // output "001"
printf( "fifth: %s", str.sub[ 4 ] ); // output "hello mum"
Note
Single quotes are stripped off the string
