Detailed Description

Provides an easy way of getting file versions short description.

This class allows access to the version information about any file, althought it is assummed that it will normally be used for your current file. Create the class, passing it the file name, including path, this will then populate the strings for the relevant vwersion information.

Currently this class is for windows only, and utilisses the Windows file version information, available by right clicking on a file , is is possible to add Linux code later, but this will get the information from within the code - yet to be written/confirmed

If the file does not contain these items the pointers will be set to zero.

An example QT use would be 

    cBncsVersion info(qApp->argv()[0]);
    qDebug(info.szVersion);
    qDebug(info.szDate);
    qDebug(info.szTime);
    qDebug(info.szDescription);

A standard windows app could use 

    szModFileName[256];
    GetModuleFileName(hInst,szModFileName,256);
    cBncsVersion info(szModFileName);
    qDebug(info.szVersion);
    qDebug(info.szDate);
    qDebug(info.szTime);
    qDebug(info.szDescription);

Operating system Notes

For Windows you will also need to link to version.lib
