BNCS Control

Description

This is an all-round control that can be used for most of the hard work of panel building. This control draws together a number of the features that are available in a number of other controls that have evolved over time.
This control can be used in Qt Designer directly but only the default presentation will appear. The full editing capabilities of this control are only available in the BNCS Visual Designer.
This control is "reskinable" in that the code required to draw it is separate from the button itself or it's settings.
Wherever possible don't explicitly set parameters - use the default or a stylesheet.

Settings

You can pass any of the following settings to this control in the format:
<parameter>=<value>
Default values (those when a new button is created or when a reset command issued) are shown in bold.
