Detailed Description

A BNCS Driver / Client Configuration Class.

This is a convenience class for loading configuration information.

Note
Please take care with paths - some are config root, some config/system

Load a single section of configuration file. The file is parsed once on load and is very efficient for reading subsequent entries.
Parameters

[in]sectionThe section to load

Note
File path is the root of the config directory (for V4 based this is config NOT config/system where dev_xxx.ini files are)

Get configuration path
Returns
Path to the configuration directory of the current system "Current System" is defined as V4 if the environment variables CC_ROOT and CC_SYSTEM are defined V3 if a ConfigPath is set in "c:\bncs_config.ini" V2 otherwise (which is the Windows directory).

int ccConfig::workstation ( void  )
static 
Get workstation number

Workstation number is a special case - this can come either from file (in more locations than one.....) or from environment variables For a V4 system the workstation number comes from the environment variable CC_WORKSTATION For a V2 or V3 system the workstation number is taken from "csi.ini" in the path specified by getDevPath()

Returns
workstation number
