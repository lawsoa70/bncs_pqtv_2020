Detailed Description

This is a simple date/time class.

Internally it uses the SYSTEMTIME and FILETIME structures - maximum extent is set by these structures.

void bncs_dateTime::toString ( char *  out,
int  len,
const char *  format = "dd MMM yyyy" 
) 
Convert to string
Parameters
[out]outWhere to write the string
[in]lenThe length of the out parameter
[in]formatThe formatting string - how you want the date/time to appear
Note
The size of out should be at least the size of the format string plus whatever is required extra for expansion of long-form data e.g. day/month names

The format string has various placeholders for date/time in various formats or can be one of the following values:

Value Meaning
BNCSDATE Date in traditional BNCS format (ApplCore)
e.g. 01 Jan 2012
BNCSTIME Time in traditional BNCS format (ApplCore)
e.g. 13:00:00
ISO Time in ISO format i.e. YYYY-MM-DDTHH:MM:SS

If it isn't one of these fixed values then it can be made up with the following formatting characters.

Note
Any vales that aren't recognised are just left intact so you can include some punctuation characters when you build your string
Placeholder Meaning
h Hours without leading zero
hh Hours with leading zero
m Minutes without leading zero
mm Minutes with leading zero
s Seconds without leading zero
ss Seconds with leading zero
z Milliseconds without leading zero
zzz Milliseconds with leading zeros
d Day without leading zero
dd Day with leading zero
ddd Day name - short form (Mon, Tue, Wed, Thu, Fri, Sat)
dddd Day name - long form (Monday etc.)
ddddd Day with suffix - e.g. 1st, 23rd
M Month without leading zero
MM Month with leading zero
MMM Month name - short form (Jan, Feb, Mar, Apr, May, Jun, Jul, Aug, Sep, Oct, Nov, Dec)
MMMM Month name - long form (January etc.)
yy Year - short form e.g. Jan 12
yyyy Year - long form e.g. Jan 2012
bncs_dateTime dt = bncs_dateTime::now(); // assuming this is 01 Feb 2012, 13:00:00
char *p;
p = dt.toString( "BNCSDATE" ); // p points to "01 Feb 2012"
p = dt.toString( "BNCSTIME" ); // p points to "13:00:00"
p = dt.toString( "ISO" ); // p points to "2012-02-01T13:00:00"
p = dt.toString( "dddd ddddd MMMM yyyy hh:mm:ss:zzz" ); // p points to "Wednesday 1st February 2012 13:00:00:000"
