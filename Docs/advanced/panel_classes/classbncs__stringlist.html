bncs_string bncs_stringlist::getNamedParam ( const bncs_string &  in) 
Extract a named parameter from a bncs_stringlist
Note
This is a Colledia Control V4.5 specific function

The purpose of this function is to extract a single <parameter>=<value> pair and return the value

Example:
device=1,offset=0,
bncs_stringlist sl( "device=123,offset=100," );
bncs_string d = sl.getNamedParam( "device" ); // d contains 123
bncs_string o = sl.getNamedParam( "offset" ); // o contains 100
Note
This isn't the most efficient function - the list has to be split up and searched for each parameter requested

bncs_string & bncs_stringlist::operator[] ( int  pos) 
Return the n'th item in the list
Parameters
positem in the list to return
Returns
item if found or zero length string if not
Example:
bncs_stringlist sl( "1,2,3,4,5," );
bncs_string s = sl[ 1 ]; // s contains "2" (the second item in the list)
From version 4.5.1 can also be used to set the value at that index
bncs_stringlist sl( "1,2,3,4,5," );
bncs_string s = sl[ 1 ]; // s contains "2" (the second item in the list)
sl[ 1 ] = "blibble";
s = sl[ 1 ]; // s contains "99" (the changed second item in the list)
sl[ 11 ] = "blibble"; // nothing happens (there is no item 11)
Note
Use as l-value only possible where an entry already exists
