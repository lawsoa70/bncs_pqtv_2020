Detailed Description

This is a STL like string handling class.

It is more fully featured than the STL class and like the Qt class QString only not quite as sophisticated.

The class members are as far as possible like the QString equivalent - so blame any inconsistancies on Trolltech and not me! A Qt developer licence is not required to use it.

It is also designed with a couple of BNCS helper functions

Constructing strings
This can be done in several different ways. Consider the following examples:
bncs_string contents( "Hello mum" );
bncs_string s1 = bncs_string( "IW ")
+ 123
+ " '" + contents + "' "
+ 456;
bncs_string s2 = bncs_string( "IW %1 '%2' %3" )
.arg( 123 )
.arg( contents )
.arg( 456 );
Both of these expressions produce the same output of:
IW 123 'hello mum' 456
S1 is scrappy and difficult to read, the quotes and spaces around the string have to be added separately which does not help.
S2 is easier to read - the "shape" of the string is already apparent in the first line, this method also allows more options for formatting the string (see the documenation for arg(int, char, unsigned int, int) ).
Splitting strings
A single function exists within the bncs_string class for splitting strings - split(). This is an equivalent to ApplCore's TD command.
An external helper class bncs_stringlist() is required to split strings into multiple parts

Parameters
ininput bncs_string
Example:

Note
Subsequent placeholders are decremented so that consecutive.arg calls may be used. Placeholders must be sequential values 1-9 (%%1 to %%9).

bncs_string s2 = bncs_string( "hello %1 %2 mum" ).arg( s1 ); // s2 contains "hello my %1 mum"
Using the % symbol in the data itself can easily confuse this mechanism - you can always build strings using + or append

int bncs_string::split ( char  delim,
bncs_string &  part1,
bncs_string &  part2 
) const 
Split a string in two

This function is intended as a convenience rather than a comprehensive split routine. "Proper" string splitting into multiple parts, with multiple delimiters is undertaken outside of this class.

Parameters
delimthe character to split on
part1the string before the first delimiter
part2the string after the delimiter - this may include more delimiter characters
Returns
index of the delimiter or -1 if no delimiter was found
Example:
bncs_string s( "param=value" );
bncs_string p1;
bncs_string p2;
s.split( '=', p1, p2 ); // p1 contains "param", p2 contains "value"
Note
If no delimiter is found neither part1 or part2 are modified - you should check the return value is >-1 to ensure that the input string was split up. If you want part1 to default to the same value as the input string if no delimiter is found then explicitly set it before calling split e.g.:
bncs_string s( "param&value" );
bncs_string p1 = s;
bncs_string p2;
s.split( '=', p1, p2 ); // p1 contains "param&value", p2 contains nothing
