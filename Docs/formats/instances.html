Overview

instances.xml contains the table of all device instances present within the system

Attribute Usage Comments
id unique identifier string  
alt_id Alternate ID for panelman to display (Optional)  
type known device type  
device device ID number  
offset base slot within device  
complex Location of the complex instance information, this is optional  
info info about the device?

DTD

<?xml version="1.0" encoding="UTF-8"?>
  <!ELEMENT instances (instance+)>

<!ELEMENT instance EMPTY>
	<!ATTLIST instance id CDATA #REQUIRED>
	<!ATTLIST instance type CDATA #REQUIRED>
	<!ATTLIST instance ref CDATA #REQUIRED>
	<!ATTLIST instance info CDATA #REQUIRED>
	<!ATTLIST instance alt_id CDATA #IMPLIED>
