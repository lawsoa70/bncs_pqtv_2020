<%@LANGUAGE="VBSCRIPT" CODEPAGE="1252"%> File Format - applications.xml

Overview

applications.xml contains the settings for each application

Each application element defines the caption, title and icon used for the app specified in the id attribute

Sample Contents - /config/applications.xml

<?xml version="1.0" encoding="UTF-8"?>
<!DOCTYPE applications SYSTEM "applications.dtd">
<applications>

DTD

<?xml version="1.0" encoding="UTF-8"?>
  <!ELEMENT applications (app+)>

<!ELEMENT app EMPTY>
  <!ATTLIST app id CDATA #REQUIRED>
  <!ATTLIST app caption CDATA #REQUIRED>
  <!ATTLIST app title CDATA #REQUIRED>
  <!ATTLIST app icon CDATA #REQUIRED>
