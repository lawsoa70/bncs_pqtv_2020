Overview

panelman.xml contains the settings for the PanelMan application

The default settings are stored for the following parameters:

Alarms - AlarmDevice, AlarmSlot and AlarmApp are used to configure the behaviour of the Alarm button

Style - the default style to use for all applications

KnobsDevice - the Knobs device ID for this system

For each user an override can be specified. For example a different style may be used or the user may need to monitor a different alarm.

Sample Contents 

<?xml version="1.0" encoding="UTF-8"?>
<!DOCTYPE users SYSTEM "panelman.dtd">
<panelman>
	<user id="default"/>
		<setting name="AlarmDevice" value="301"/>
		<setting name="AlarmSlot" value="205"/>
		<setting name="AlarmApp" value="almover"/>
		<setting name="Style" value="bncs_style3"/>
		<setting name="KnobsDevice" value="303"/>
	</user>
	<user id="chris">
		<setting name="AlarmApp" value=""/>
	</user>
	<user id="satops">
		<setting name="AlarmDevice" value="301"/>
		<setting name="AlarmSlot" value="322"/>
		<setting name="AlarmApp" value="almsat"/>
	</user>
</panelman>

DTD

<?xml version="1.0" encoding="UTF-8"?>
  <!ELEMENT panelman (user+)>

<!ELEMENT user (setting+)>
  <!ATTLIST user name CDATA #REQUIRED>
  <!ATTLIST user value CDATA #REQUIRED>
