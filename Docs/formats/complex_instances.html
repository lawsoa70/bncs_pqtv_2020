Overview

The config\complex_instances folder contains files detailing all the complex device instances present within the system

Attribute Usage Comments
id unique identifier string for the parameter  
device Device on which the real parameter is located  
param Parameter on which the real device  

Sample Contents - /config/complex_instances/complex_dev_instances.xml

<?xml version="1.0" ?>
<complex_dev_instances>
    <instance id="complex_arc1">
        <param id="Input-gpi" device="gpi_crate_1" param="GPI Input 01"/>
        <param id="Ref-gpi" device="gpi_crate_1" param="GPI Input 02"/>
        <param id="PSU-gpi" device="gpi_crate_1" param="GPI Input 03"/>
    </instance>
    <instance id="complex_arc2">
        <param id="Input-gpi" device="gpi_crate_1" param="GPI Input 04"/>
        <param id="Ref-gpi" device="gpi_crate_1" param="GPI Input 05"/>
        <param id="PSU-gpi" device="gpi_crate_1" param="GPI Input 06"/>
    </instance>
</complex_dev_instances>
