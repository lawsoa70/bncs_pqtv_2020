Overview

devicetypes.xml contains the table of all device types present within the system

The detail for each type is stored in the corresponding devicetype file in /config/devicetypes/???.xml

Attribute Usage Comments
id unique identifier string  
family family to which type belongs  

Sample Contents - /config/devicetypes.xml

<?xml version="1.0" ?>
<!DOCTYPE devicetypes SYSTEM "devicetypes.dtd">
<devicetypes>
	<type id="SCP1121" family="GLUE_SIGMON"/>
	<type id="V1642" family="GLUE_PROC"/>
	<type id="IQDSIG" family="GLUE_INSERTER"/>
</devicetypes>

DTD

<?xml version="1.0" encoding="UTF-8"?>
<!ELEMENT devicetypes (type+)>
<!ELEMENT type EMPTY>
<!ATTLIST type
	id CDATA #REQUIRED
	family CDATA #REQUIRE
>
