Overview

The config/drivers folder contains the config file for each of the driver instances present within the system

The example shown below would be called config/drivers/dev_230.xml or config/drivers/rollcall.xml?

DTD

<?xml version="1.0" encoding="UTF-8"?>

<!ELEMENT driverconfig (paramgroup+)>
	<!ATTLIST driverconfig id CDATA #REQUIRED>
	<!ATTLIST driverconfig type CDATA #REQUIRED>
	<!ATTLIST driverconfig deviceid CDATA #REQUIRED>

<!ELEMENT paramgroup (param+)>
	<!ATTLIST paramgroup id CDATA #REQUIRED>

<!ELEMENT param EMPTY>
	<!ATTLIST param id CDATA #REQUIRED>
	<!ATTLIST param type CDATA #REQUIRED>
	<!ATTLIST param value CDATA #REQUIRED>
